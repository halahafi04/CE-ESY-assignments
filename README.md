# Assignment 1 - Circular Buffer in C

This repository contains the C programming implementation of a Circular Buffer as required for the CE-ESY course.

## Files
- Assignment1/prog.c: The main source code implementing the circular buffer operations.

## Features
- Implemented Circular Buffer with Initialization, Write, and Read functions.
- Correctly handles Buffer Overflow and Buffer Underflow conditions.
- Appends the string "CE-ESY" to the input name and processes it character by character.
- Interactive user input with the ability to repeat the operation.

## How to Compile and Run
1. Open your terminal or an online compiler (such as Programiz or similar).
2. Compile the C code using gcc:
   `bash
   gcc prog.c -o prog
